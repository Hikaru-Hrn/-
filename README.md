# Парсер выписки из Тбанка

Этот код выполняет простую задачу: Перевести pdf формат выписки о движении средств в csv.


## Инструкция
Установите доп.библеотеку:
```
pip install pdfplumber
```

А затем просто запустите код, поместив файл с выпиской в ту же директорию. При надобности измените имя загружаемого файла. А затем запустите программу:
```
python pdf_converter.py
```
